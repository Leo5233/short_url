安裝說明

使用 terminal，將專案下載至本機電腦

    git clone https://github.com/Leo5233/short_url.git

進入存放此專案的資料夾

    cd short_url

使用npm安裝 express和express-handlebars

    npm install

安裝 nodemon 套件，有安裝過可略過

    npm.install nodemon

執行專案

    npm run dev

開啟瀏覽器輸入網址localhost:3000

功能列表

    可以產生縮網址

    限制輸入須為有效網址格式

    點擊畫面下方Shorten鈕即跳出縮網址

    點擊畫面下方Copy鈕即可複製網址
